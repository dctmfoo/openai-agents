# Admin (Tauri v2)

Minimal Tauri v2 desktop app scaffold for the admin UI. The frontend is a tiny static page in `frontend/`.

## Prerequisites

- Rust toolchain (stable)
- Node.js + pnpm

## Run (dev)

```bash
cd apps/admin
pnpm install
pnpm dev
# or: pnpm tauri dev
```

## Build (release)

```bash
cd apps/admin
pnpm build
```
